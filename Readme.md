# Backend

```
run mysql on port 3306
npm install
npm run dev
```

Using yarn

```
yarn
yarn run dev
```

### Using docker

```
npm run docker:build
npm run docker:start
```

# Frontend

## Requirements
    - `node v16.14.2`

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
